Backend Integration

At a first glance

Because JSpective fetches all its data via AJAX requests anyway, integrating dynamic sources should be straight forward.
Any http server will do the job. You can feed standard html pages or any XML into JSpective.

Sources and contexts

Questions about the user interface's behaviour will arise, when data is to be symbolized and displayed.
  • What data - and in which context - is interesting being explored in a 3D browser?
  • How should efficient user interface widgets look alike for that data and context?
  • What 3D widget behaviour would add substantial value to a 2D world?
