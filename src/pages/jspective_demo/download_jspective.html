Downloads

Please use the mirroring services of Sourceforge to download JSpective:

The latest versions of JSpective are always published as tgz-archives.
Go to the JSpective Download Page (at Sourceforge.net).
  • Just unpack the tgz-archive (will create folder "jspective_vXXX/").
  • Open jspective_vXXX/src/index.html in a recent webbrower.
  • That's it.

CVS Repository

Use your web browser to access the source read-only:
JSpective CVS repository (hosted by Sourceforge.net).

Or use a CVS client for regular downloads of the freshest files.
